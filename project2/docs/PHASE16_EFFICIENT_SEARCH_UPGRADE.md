# Phase 16 Efficient Search Upgrade

This version improves the top global search field.

## Improvements
- Search now works with actual values such as `Chocolate`, `Lot's O' Nuts`, `Texas`, `Houston`, `CHO-MIL`, and `Wonka`.
- Search also understands field/category words such as `factory`, `factories`, `product`, `products`, `division`, `region`, `state`, `city`, `customer`, `order`, and `postal`.
- Multi-word search uses smarter matching, so queries like `wonka chocolate` are handled more effectively.
- Search diagnostics now include matching rows and unique matches.
- Sidebar version updated to v16.0.
