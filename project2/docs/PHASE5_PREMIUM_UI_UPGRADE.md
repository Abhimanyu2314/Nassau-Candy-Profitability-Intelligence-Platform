# Phase 5 Premium BI Upgrade

This version upgrades the Streamlit project from a tab-based dashboard into an enterprise-style BI control tower.

## Added
- Left sidebar enterprise navigation
- Dark executive theme
- Glassmorphism KPI cards
- Hover-style KPI effects
- Executive overview page
- Dedicated product, division, cost, Pareto, factory, risk, recommendation, advanced analytics, and reports pages
- Interactive US factory profitability map
- CFO-style profit waterfall
- Margin heatmaps
- Industrial decision matrix
- Report and export center

## Run
```bash
pip install -r requirements.txt
streamlit run app.py
```
