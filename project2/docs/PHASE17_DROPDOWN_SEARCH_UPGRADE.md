# Phase 17 Dropdown Search Upgrade

This version replaces the plain top search text field with a searchable dropdown selector.

## Improvements
- Global search is now a dropdown with built-in type-to-search.
- Search suggestions are grouped visually by category:
  - Product Name
  - Product ID
  - Division
  - Factory
  - Region
  - State
  - City
- Smart shortcuts are included:
  - Show all factories
  - Show all products
  - Show all divisions
  - Show all regions
- Selected dropdown values still filter the entire analytics pipeline.
- Settings page diagnostics continue to show the active search and matching rows.
