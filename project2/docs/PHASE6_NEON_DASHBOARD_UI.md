# Phase 6 Neon Executive Dashboard UI Upgrade

This version redesigns the Streamlit interface to match a premium dark executive dashboard style:

- Purple, pink, and orange neon visual language
- Glassmorphism KPI cards
- Angled top command bar with search/user area
- Dark left navigation rail styling
- Neon borders, hover animations, and high-contrast analytics cards
- Plotly dark theme with custom neon color palette

The analytics engine remains unchanged, so all profitability, margin-risk, factory intelligence, Pareto, recommendations, and export features continue working.
