NASSAU CANDY PROFITABILITY INTELLIGENCE PLATFORM - RUN GUIDE

BEST WAY TO RUN ON WINDOWS:
1. Extract this ZIP.
2. Open the extracted folder.
3. Double-click RUN_APP.bat.

MANUAL METHOD:
1. Open PowerShell inside this folder.
2. Run:
   pip install -r requirements.txt
3. Run:
   streamlit run app.py --server.headless false --server.address localhost --server.port 8501
4. Open this exact URL in your browser:
   http://localhost:8501

IMPORTANT:
Do not open the External URL such as http://152.xx.xx.xx:8501.
Use only http://localhost:8501 or http://127.0.0.1:8501 on your PC.

If the browser shows blank:
- Stop Streamlit with Ctrl+C
- Run: streamlit cache clear
- Run RUN_APP.bat again
